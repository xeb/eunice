### High-Synergy Combinations
- **memory + web**: Research agents that build persistent knowledge graphs
- **filesystem + grep + shell**: Code analysis and maintenance agents
- **fetch + memory**: Web monitoring and change tracking agents
- **filesystem + text-editor**: Documentation and writing agents

### Underexplored Combinations
- memory + shell: Stateful command orchestration
- grep + web: Cross-referencing local code with online documentation
- fetch + filesystem: Web archiving and local mirroring
