A P I S P E C

GET /api/status

Returns server status, model info, and available tools.

Example Response 
{
  "version": "0.2.29",
  "model": "gemini-3-pro-preview",
  "provider": "Gemini",
  "mode": "dmn",
  "agent": null,
  "mcp_servers": ["shell", "filesystem"],
  "tools_count": 12
}

POST /api/query

Submit a query. Returns Server-Sent Events (SSE) stream with tool calls, results, and response.

SSE Event Types 
event: thinking
data: {"type":"thinking","elapsed_seconds":1}

event: tool_call
data: {"type":"tool_call","name":"shell_execute","arguments":"{\"command\":\"ls\"}"}

event: tool_result
data: {"type":"tool_result","name":"shell_execute","result":"file1.txt\nfile2.txt","truncated":false}

event: response
data: {"type":"response","content":"The directory contains 2 files."}

event: done
data: {"type":"done"}

POST /api/cancel

Cancel the currently running query.

Example Response 
{"cancelled": true}
